# Buscador de Recetas - Gourmet Go

Sprint 1: Maquetación y Prototipo Visual

## Descripción

Prototipo visual para el Buscador de Recetas de Gourmet Go.

## Historias de Usuario Implementadas

### HU-01: Interfaz Principal de Búsqueda
- Título principal "Buscador de Recetas"
- Campo de búsqueda con placeholder descriptivo
- Botón de búsqueda estilizado con Bootstrap
- Formulario responsivo y centrado

### HU-02: Visualización de Recetas de Ejemplo
- Sección de resultados con 6 tarjetas de ejemplo
- Cada tarjeta incluye imagen, título y botón
- Uso del componente card de Bootstrap
- Contenido estático (hard-coded)

### HU-03: Experiencia en Dispositivos Móviles
- Diseño completamente responsivo
- Grilla de 3 columnas en desktop
- Apilamiento vertical en móviles
- Sistema de grillas de Bootstrap implementado

## Instalación y Uso

1. Clonar el repositorio
2. Abrir index.html en el navegador
3. No requiere instalaciones.
